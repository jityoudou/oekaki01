
# P5test
## これは何？
Make Faire Tokyo 2021 に向けたP5jsなどなどの練習用環境

## 始め方
node, npmをインストールした状態で↓を実行、 その後`localohst:4200`へアクセス
```
npm ci
npm run start
```

動作確認済バージョン
```
node: v10.19.0
npm: 6.14.4
```

## 使用しているフレームワーク

 * Anguler セッション管理やAPIへの接続など
 * RxJS 各種非同意処理
 * p5js 画面表示関係
